# SM Plugin Starter

Wordpress plugin starter for development

## To start using

Follow these steps: 

1. Set variables in config.json
2. Set plugin name in package.json
3. Set config in ./sm-plugin-starter/plugin-config.php
4. Run `npm run initialize`
